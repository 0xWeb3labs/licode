Roadmap

Overview of Lynckia's future enhancements.

1. Current Projects

Lynckia offers video, audio and data communication to several projects today!

Lynckia now is fully functional and you can try it from last versions of Google Chrome. Actually weÂ´re currently using Lynckia in several projects at the University:

  • MCS5:

    A national project with Telefonica to demonstrate new WebRTC features in desktop and mobile platforms. In this project we will show Lynckia in action in different scenarios: social TV, games and video-surveillance.

  • FI-CONTENT:

    Lynckia is being implemented in a SmartCity application to show tourist videos to several users. Lycnkia will be integrate inside FIWare infrastructure to serve real-time communications and support real-time synchronization among clients. These clients will be connected from desktop and mobile clients just using the web browsers.

2. Future Enhancements

Lynckia is growing very fast, with new enhancements and features.

We are currently working to develop the next features in the short term:

  • Support to more RTCP packets.
  • Support to other browsers: Ericsson Bowser and Mozilla Firefox.
  • Include external streams from RTMP, RTSP, raw RTP, and so on
  • Improve monitorization and notifications about rooms, users and streams
  • Support for non-browser based clients.
