Installation

Guidelines to easily install Lynckia in your environment.

  • Requirements
  • Clone the code
  • Dependencies
  • Lynckia
  • Basic Example
  • Test it!

1. Requirements

Install Lynckia on Ubuntu Linux.

Ubuntu 12.04 LTS

Our recommendation is to install it in a clean installation.

Mac OS X

It has been tested with Mountain Lion.

2. Clone Lynckia code

You first need to clone our code from github.

You need to have git installed in your computer.

In ubuntu use "sudo apt-get install git".

In Mac OS X you need to have installed Xcode Command Line Tools. Also you can install git with homebrew.

Now clone the repository. 
git clone https://github.com/ging/lynckia.git

All scrips used below are included in this code cloned from github.

3. Install Dependencies

Now you need to install some dependencies to use Lynckia.

This step depends on the type of Operating System your are using.

Ubuntu 12.04 LTS. 
./lynckia/scripts/installUbuntuDeps.sh
Mac OS X Mountain Lion. 
./lynckia/scripts/installMacDeps.sh

4. Install Lynckia

This step will install all Lynckia components.

These components are: Erizo, Erizo API, Erizo Controller and Nuve.

You will only need to execute these commands. They do not depends on the Operating System. 
./lynckia/scripts/installErizo.sh
./lynckia/scripts/installNuve.sh

5. Install the Basic Example

This step will install a basic test example.

You will only need to execute the next command. It does not depends on the Operating System. 
./lynckia/scripts/installBasicExample.sh

8. Test Lynckia!

At this time you have already installed in your computer Lynckia and a basic videoconference example to test it. Let's run them.

This step will initialize all Lynckia components. 
./lynckia/scripts/initLynckia.sh
This step will initialize the basic example. 
./lynckia/scripts/initBasicExample.sh

Now you can connect to localhost:3004 and test your basic videoconference example!
