Installation

Guidelines to easily install Lynckia in your environment.

1. Requirements

Install Lynckia on Ubuntu Linux.

Ubuntu 12.04 LTS

Our recommendation is to install it in a clean installation.

Mac OS X

it has been tested with Mountain Lion.

2. Install Dependencies

You first need to install dependencies if you want to use Lynckia.

This step depends on the type of Operating System your are using.

Ubuntu 12.04 LTS. 
bash <(curl -s http://chotis2.dit.upm.es/installDepsUbuntu.sh)
Mac OS X Mountain Lion. 
bash <(curl -s http://chotis2.dit.upm.es/installDepsMac.sh)

3. Install Erizo components

This step will install all Erizo components.

These components are: Erizo, Erizo API and Erizo Controller.

You will only need to execute the next command. It does not depends on the Operating System. 
bash <(curl -s http://chotis2.dit.upm.es/installErizo.sh)

4. Install Nuve

This step will install Nuve.

You can check the source code here.

You will only need to execute the next command. It does not depends on the Operating System. 
bash <(curl -s http://chotis2.dit.upm.es/installNuve.sh)

5. Install the Basic Example

This step will install a basic test example.

You will only need to execute the next command. It does not depends on the Operating System. 
bash <(curl -s http://chotis2.dit.upm.es/installBasicExample.sh)

6. Run Lynckia Services

This step will initialize all Lynckia components.

You will only need to execute the next command. It does not depends on the Operating System. 
bash <(curl -s http://chotis2.dit.upm.es/initLynckia.sh)

7. Run the basic Example

This step will initialize the basic example.

You will only need to execute the next command. It does not depends on the Operating System. 
bash <(curl -s http://chotis2.dit.upm.es/initBasicExample.sh)
